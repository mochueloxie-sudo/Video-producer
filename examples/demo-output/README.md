# Demo Output

This directory contains sample output generated by the full pipeline (`command: "all"`) using [`../full_variant_test.md`](../full_variant_test.md) as input.

**Theme**: `terminal-green` (auto-selected)
**Pages**: 11 slides

## Files

| File | Description |
|------|-------------|
| `presentation.html` | Primary deck: iframe loads each `page_XXX.html` (hover + entrance animations; regenerate to match current pipeline) |
| `presentation_static.html` | PNG-based carousel (same frames as PDF); present when regenerated with current Step 6 |
| `presentation.pdf` | PDF document — one 1920×1080 slide per page |
| `outline.md` | Content outline with page numbers, variants, and keywords |
| `script.md` | Full narration script, organised by slide |

## How it was generated

```bash
echo '{"command":"all","source":"./examples/full_variant_test.md","format":["pdf","html"],"output_dir":"./output"}' | node executor.js
```
